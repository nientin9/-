# Python Web Crawler
## Selenium自動測試-觀看廣告兌換抽抽樂活動抽獎機會
Step 1. 輸入登入帳號後的cookie  
(cookie一般都存在一定的時效，不同網站的cookie失效時間不同，所以當cookie失效之後需要重新手動登入獲取！)

Step 2. 執行Selenium

執行過程中，網頁可能彈跳出錯誤視窗，需重新執行Selenium  
錯誤訊息包含:  
* 發生錯誤，請重新嘗試(1)
* 廣告能量補充中 請稍後再試。
* 已移除廣告。(這個廣告使用了太多裝置資源，因此 Chrome 已將其移除。(兌換截止))
---
關於每一個抽抽樂活動:  
★ 使用觀看廣告可免費兌換 1 次抽獎機會。  
★ 1 天上限最多可觀看 10 次廣告。
